# cLess

## Setup 
add this to /etc/hosts
```
127.0.0.1       golang.cless.cloud
127.0.0.1       python.cless.cloud
127.0.0.1       java.cless.cloud
127.0.0.1       nodejs.cless.cloud
127.0.0.1       rust.cless.cloud
```

## Run Server
go run main

## Test or Browse
curl http://rust.cless.cloud/
